############ AWS KINESIS ##############

> we can use Amazon Kinesis Data Streams to collect and PROCESS LARGE STREAMS OF DATA RECORDS IN REAL TIME. 
- we can create data-processing app, known as Kinesis Data Streams app. 
- A typical Kinesis Data Streams app reads data from a data stream as data records. 
- These app can use the Kinesis Client Library, and they can run on Amazon EC2 instances. 
- You can send the processed records to dashboards, use them to generate alerts, dynamically change pricing and advertising strategies, or send data to a variety of other AWS services
- we can continuously add various types of data such as clickstreams, application logs, and social media to an Amazon Kinesis data stream from hundreds of thousands of sources. 
- Within seconds, the data will be available for Amazon Kinesis App to read and process from the stream.
