########## REST Assured #############
> It can be used to test XML as well as JSON based web services. 
> It can be integrated with JUnit and TestNG frameworks for writing test cases
> It uses the builder pattern to create requests, set headers, parse the response and then match them with expected data. 
> It uses Hamcrest Matchers for comparing actual response with the expected response.

> One of the powerful features of REST assured is the support of XML Path and JSON Path syntax to check specific elements of the response data.
